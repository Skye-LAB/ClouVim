## Installation

First, clone this repo to your `.config/vim`

```bash
git clone https://github.com/Skye-LAB/ClouVim.git ~/.config/vim
```

Then make a symlink to `~/.vimrc`

```bash
ln -s ~/.config/vim/.vimrc ~/.vimrc
```
